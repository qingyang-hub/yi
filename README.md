# 英语单词记忆系统

这是一个帮助学习者高效记忆专四核心词汇的在线学习系统。系统提供随机单词和默写两种学习模式，让用户能够灵活地进行单词记忆和复习。

## 功能特点

- 随机单词模式：随机展示单词，帮助用户进行记忆
- 默写模式：通过默写练习加深记忆
- 分组学习：单词按照不同主题进行分组，便于系统学习
- 响应式设计：支持各种设备访问

## 部署说明

1. 克隆本仓库到本地
2. 确保所有文件（HTML、CSS、JavaScript）都在正确的位置
3. 将仓库推送到GitHub
4. 在仓库设置中启用GitHub Pages
5. 访问生成的网站地址
